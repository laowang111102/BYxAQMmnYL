## 项目概述 java264


## 技术栈

- 前端框架：Vue3
- UI 库：Element Plus
- 构建工具：Vite
- 状态管理：Pinia
- 路由管理：Vue Router
- 图表库：ECharts

## 功能模块

系统包含以下 12 个页面：

1. 登录页
2. 注册页
3. 仪表盘首页
4. 水果分类管理
5. 水果库存管理
6. 订单管理
7. 用户管理
8. 角色管理
9. 权限管理
10. 个人中心
11. 系统设置
12. 错误页面

共包含 15 个自定义组件。

## 系统亮点

- 分页功能
- Pinia 实现状态管理
- ECharts 数据可视化
- 父子组件通信（Prop、Emit）
- 指令支持（v-if、v-for、v-on、v-model、v-bind）
- 计算属性（computed）和侦听器（watch）
- RBAC 权限控制系统

## 系统角色

- 系统管理员：负责系统运维、用户管理、角色权限分配等。
- 普通用户：负责日常业务操作，如订单管理、库存管理等。

## 运行环境

- Node.js 版本：建议使用最新稳定版
- 浏览器：主流浏览器（Chrome、Firefox、Safari、Edge 等）

## 部署步骤

1. 克隆项目到本地
2. 安装依赖（`npm install` 或 `yarn`）
3. 运行开发服务器（`npm run dev` 或 `yarn dev`）
4. 打包生产环境代码（`npm run build` 或 `yarn build`）
5. 将打包后的代码部署至服务器

本项目采用 Vue3 最新技术栈，结合 Element Plus UI 库，为开发者提供了一套易于上手和定制的后台管理系统模板。系统的功能模块齐全，且具备完善的权限控制系统，适用于学习、二次开发或作为企业内部管理系统使用。
